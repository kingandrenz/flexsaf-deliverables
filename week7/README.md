# CLI To-Do List App (Function-Based)

## Project Overview
This project is a simple **Command-Line Interface (CLI) To-Do List application built with Python**.  
It allows users to manage their daily tasks directly from the terminal.

The application demonstrates the use of **Python functions, modules, loops, and built-in functions** while organizing code into reusable components.

---

## Features
The program allows the user to:

- Add a new task
- View all tasks
- Mark a task as completed
- Delete a task
- View task statistics
- Exit the application

The application runs continuously until the user chooses to exit.

## Concepts Demonstrated

This project demonstrates the following Python concepts:

- Defining and calling reusable functions
- Using parameters and arguments
- Returning values from functions
- Organizing code into modules
- Importing functions using `import` and `from ... import`
- Using built-in functions such as:
  - `len()`
  - `enumerate()`
  - `sum()`
- Iterating through lists using loops
- Handling invalid user input with `try` and `except`

## How to Run the Program

1. Make sure **Python is installed** on your computer.
2. Download or clone this project.
3. Open a terminal inside the project folder.
4. Run the program with:

       python main.py or python3 main.py


5. Follow the menu instructions displayed in the terminal.
