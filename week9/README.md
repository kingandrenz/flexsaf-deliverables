# Student Scores CSV Processor

## 📘 Overview
This project demonstrates how to:
- Write student data (names and scores) into a CSV file.
- Read the data back from the CSV.
- Calculate the class average score.
- Identify the top-performing student.

## ✨ Features
- Uses Python's built-in `csv` module for file handling.
- Converts scores to integers for accurate calculations.
- Skips the header row when reading back data.
- Prints both the class average and the top performer.

## ⚙️ Requirements
- Python 3.x
- No external libraries required (uses standard library only)

## 🚀 Usage
1. Clone or download this repository.
2. Ensure you have Python installed (`python --version`).
3. Run the script:
   ```bash
   python3 filehandler.py
